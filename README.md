# Single Page Resume Website (HTML Only)

This project is a **single-page resume website** built using **only HTML**, as part of an academic assignment.  
No CSS, JavaScript, or external libraries are used. The focus is on **HTML structure, semantics, and layout clarity**.

The resume layout is inspired by the sample provided in the assignment and includes personal information filled with my own data.
